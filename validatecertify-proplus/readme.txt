== Description ==
Con ValidateCertify, puedes garantizar la autenticidad y veracidad de los certificados emitidos, brindando confianza a tus alumnos y a aquellos que los validen. Simplifica el proceso de verificación y mejora la experiencia de tus usuarios de forma rápida, simple y sin necesidad de cambiar de plataforma o web para realizarlo.

Crear la validación de tus certificados es simple, solo debes cargar la base de tus certificados en nuestro plugin, este incluye **Nombres, Apellidos, Curso, Fecha y Código de Certificado.**


###Características de ValidateCertify - Free###


- Agrega certificados de forma manual a tu base de datos.
- Edita los certificados creados con el código de certificado.
- Visualiza la base de datos de tus certificados.
- Utiliza un ShortCode para añadir en cualquier página la opción de validación.


##Pasa al nuevo nivel con ValidateCertify Pro##


###Características de [ValidateCertify - Pro](https://systenjrh.com/plugin-validatecertify/)###


- Agrega certificados de forma masiva, hasta 1000 certificados de una sola carga.
- Exporta tu base de datos en cualquier momento.
- Detecta automáticamente los códigos duplicados y solo sube los nuevos certificados.
- Elimina tu base completa.
- Visualiza la base de datos de tus certificados.
- Utiliza un ShortCode para añadir en cualquier página la opción de validación.

== Installation ==

**Desde tu panel de WordPress**
1. Instale el complemento a través de la pantalla de complementos de WordPress directamente.
2. Active ValidateCertify - Pro desde su página de Complementos.

**De WordPress.org**
1. Busque y descargue ValidateCertify
1. Cargue la carpeta del complemento en el directorio `/wp-content/plugins/`
2. Active el complemento a través de la pantalla 'Complementos' en WordPress

**From your WordPress dashboard**
1. Install the plugin through the WordPress plugins screen directly.
2. Activate ValidateCertify - Pro from your Plugins page.

**From WordPress.org**
1. Search and download ValidateCertify
1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
